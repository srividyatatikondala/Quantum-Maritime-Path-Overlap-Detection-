üìç Phase 1: Ideation and Validation

Problem Statement

In maritime navigation, multiple ships travel through defined lanes. If two or more ships occupy the same lane at the same time, a **collision risk** arises. Traditional systems check lane occupancy classically, but we explore quantum computing to detect overlaps faster by exploiting quantum superposition and parallelism.

Objective

  • Represents each ship‚Äôs possible lane choice in **superposition**.
  • Detects if any two ships choose the same lane (**collision**).
  • Flags overlap probability using a quantum oracle.

Validation

The solution leverages:

  • **One-hot encoding** $\rightarrow$ ensures one lane per ship in superposition.
  • **Oracle marking** $\rightarrow$ checks lane overlaps efficiently.
  • **Measurement** $\rightarrow$ determines collision existence probability.

Circuit Components

Component Description
**Ship Registers** Each ship represented by $N_{lanes}$ qubits, one-hot encoded.
**Ancilla Qubits** Used to compare all ${N_{ships} \choose 2}$ ship pairs.

üß© Phase 4: Solved Problem Explanation

Aspect Classical Method Quantum Method
**Lane combinations** Checked one-by-one All possible combinations evaluated simultaneously
**Collision detection** Requires $N^2$ pairwise comparisons Achieved in one oracle operation through superposition
**Efficiency** $O(N^2)$ (or $O(N \log N)$ with optimized classical algorithms) Quantum parallelism ($O(1)$ check across all states)
**Real-world use** Large fleet route planning Quantum traffic management simulation

How Problem is Solved

  1. **Encoding:** Each ship's possible position (lane) is encoded in a **one-hot superposition**, ensuring all valid scenarios are present.
  2. **Detection:** The **Oracle** identifies, in parallel, where any two ships occupy the same lane.
  3. **Measurement:** Measurement collapses the quantum state and outputs a single bit probability:
    • **1** $\rightarrow$ Collision Detected
    • **0** $\rightarrow$ No Collision

Thus, the problem of path overlap detection is efficiently solved using quantum logic rather than exhaustive classical computation.
